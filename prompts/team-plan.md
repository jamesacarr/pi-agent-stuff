## Workflow

1. **Spawn** a scout and a planner.
2. **Scout**: Send the scout to find all code relevant to the task.
3. **Plan**: Send the planner the scout's findings and the task description. Wait for a concrete plan.
4. **Report** the plan to the user. Do not execute it.
5. **Dismiss** all team members when done.

## Rules

- Never summarise scout findings when sending to the planner — the planner needs full code context to produce accurate steps.
- Do not implement anything — the user wants a plan to review before committing to execution.
